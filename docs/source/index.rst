WNAE: Wasserstein Normalized Autoencoder
========================================

The Wasserstein Normalized Autoencoder is an unsupervised ML algorithm for anomaly detection.
It was developed in the context of a recent CMS search, see https://cds.cern.ch/record/2911111.
